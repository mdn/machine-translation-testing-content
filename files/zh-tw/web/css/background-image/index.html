{{CSSRef}}

background-image CSS屬性在一個元素上設置一個或多個背景圖像。

{{EmbedInteractiveExample("pages/css/background-image.html")}}
該交互式示例的源代碼存儲在GitHub存儲庫中。如果您想為交互式示例項目做出貢獻，請克隆https://github.com/mdn/interactive-examples並向我們發送請求請求。

背景圖像是在彼此堆疊的上下文圖層上繪製的。指定的第一層將被繪製為最接近用戶。

然後，將元素的邊界繪製在它們的頂部，然後{{cssxref("background-color")}}被畫在他們下面。圖像相對於框及其邊框的繪製方式由{{cssxref("background-clip")}}和{{cssxref("background-origin")}}CSS屬性。

如果無法繪製指定的圖像（例如，當無法加載由指定URI表示的文件時），瀏覽器將按照none值進行處理。

注意：即使圖片不透明並且在正常情況下也不會顯示顏色，Web開發人員也應始終指定一個{{cssxref("background-color")}}。如果無法加載圖像（例如，當網絡中斷時），背景顏色將用作後備。

句法

每個背景圖片都指定為關鍵字none或{{cssxref("<image>")}}值。

要指定多個背景圖像，請提供多個值，以逗號分隔： 

background-image:
  linear-gradient(to bottom, rgba(255,255,0,0.5), rgba(0,0,255,0.5)),
  url('https://mdn.mozillademos.org/files/7693/catfront.png');

價值觀

none
是表示沒有圖像的關鍵字。
<image>
是一個{{cssxref("<image>")}}表示要顯示的圖像。由於支持多個背景，因此可以有多個，中間用逗號分隔。

無障礙問題

瀏覽器不向輔助技術提供有關背景圖像的任何特殊信息。這主要對屏幕閱讀器很重要，因為屏幕閱讀器不會宣布其存在，因此不會向用戶傳達任何信息。如果圖像包含對於理解頁面的總體目的至關重要的信息，則最好在文檔中進行語義描述。

例子

分層背景圖片

請注意，星形圖像是部分透明的，並且位於貓圖像上。

的HTML 

<div>
  <p class="catsandstars">
    This paragraph is full of cats<br />and stars.
  </p>
  <p>This paragraph is not.</p>
  <p class="catsandstars">
    Here are more cats for you.<br />Look at them!
  </p>
  <p>And no more.</p>
</div>

的CSS 

p {
  font-size: 1.5em;
  color: #FE7F88;
  background-image: none;
  background-color: transparent;
}

div {
  background-image:
      url("https://mdn.mozillademos.org/files/6457/mdn_logo_only_color.png");
}

.catsandstars {
  background-image:
      url("https://mdn.mozillademos.org/files/11991/startransparent.gif"),
      url("https://mdn.mozillademos.org/files/7693/catfront.png");
  background-color: transparent;
}
