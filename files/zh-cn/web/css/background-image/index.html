{{CSSRef}}

background-image CSS属性在一个元素上设置一个或多个背景图像。

{{EmbedInteractiveExample("pages/css/background-image.html")}}
该交互式示例的源代码存储在GitHub存储库中。如果您想为交互式示例项目做出贡献，请克隆https://github.com/mdn/interactive-examples并向我们发送请求请求。

背景图像是在彼此堆叠的上下文图层上绘制的。指定的第一层将被绘制为最接近用户。

然后，将元素的边界绘制在它们的顶部，然后{{cssxref("background-color")}}被画在他们下面。图像相对于框及其边框的绘制方式由{{cssxref("background-clip")}}和{{cssxref("background-origin")}}CSS属性。

如果无法绘制指定的图像（例如，当无法加载由指定URI表示的文件时），浏览器将按照none值进行处理。

注意：即使图片不透明并且在正常情况下也不会显示颜色，Web开发人员也应始终指定一个{{cssxref("background-color")}}。如果无法加载图像（例如，当网络中断时），背景颜色将用作后备。

句法

每个背景图片都指定为关键字none或{{cssxref("<image>")}}值。

要指定多个背景图像，请提供多个值，以逗号分隔： 

background-image:
  linear-gradient(to bottom, rgba(255,255,0,0.5), rgba(0,0,255,0.5)),
  url('https://mdn.mozillademos.org/files/7693/catfront.png');

价值观

none
是表示没有图像的关键字。
<image>
是一个{{cssxref("<image>")}}表示要显示的图像。由于支持多个背景，因此可以有多个，中间用逗号分隔。

无障碍问题

浏览器不向辅助技术提供有关背景图像的任何特殊信息。这主要对屏幕阅读器很重要，因为屏幕阅读器不会宣布其存在，因此不会向用户传达任何信息。如果图像包含对于理解页面的总体目的至关重要的信息，则最好在文档中进行语义描述。

例子

分层背景图片

请注意，星形图像是部分透明的，并且位于猫图像上。

的HTML 

<div>
  <p class="catsandstars">
    This paragraph is full of cats<br />and stars.
  </p>
  <p>This paragraph is not.</p>
  <p class="catsandstars">
    Here are more cats for you.<br />Look at them!
  </p>
  <p>And no more.</p>
</div>

的CSS 

p {
  font-size: 1.5em;
  color: #FE7F88;
  background-image: none;
  background-color: transparent;
}

div {
  background-image:
      url("https://mdn.mozillademos.org/files/6457/mdn_logo_only_color.png");
}

.catsandstars {
  background-image:
      url("https://mdn.mozillademos.org/files/11991/startransparent.gif"),
      url("https://mdn.mozillademos.org/files/7693/catfront.png");
  background-color: transparent;
}
