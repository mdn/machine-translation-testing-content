JavaScriptは、Webサイトに双方向性を追加するプログラミング言語です。これは、ゲーム、ボタンが押されたときの応答の動作、またはフォームへのデータ入力で発生します。ダイナミックなスタイリング。この記事は、JavaScriptの使用を開始し、何が可能かについての理解を深めるのに役立ちます。

JavaScriptとは何ですか？

{{Glossary("JavaScript")}} （略して「JS」）は本格的です{{Glossary("Dynamic programming language", "dynamic programming language")}}Webサイトに双方向性を追加できます。これは、Brendan Eich（Mozillaプロジェクト、Mozilla Foundation、およびMozilla Corporationの共同創設者）によって発明されました。

JavaScriptは用途が広く、初心者にも優しいです。より多くの経験を積むと、ゲーム、アニメーションの2Dおよび3Dグラフィックス、包括的なデータベース駆動型アプリなどを作成できるようになります。

JavaScript自体は比較的コンパクトですが、非常に柔軟性があります。開発者は、コアJavaScript言語の上にさまざまなツールを作成し、最小限の労力で膨大な量の機能のロックを解除しています。これらには以下が含まれます：

  • ブラウザアプリケーションプログラミングインターフェイス（{{Glossary("API","APIs")}} ）Webブラウザーに組み込まれ、HTMLの動的な作成やCSSスタイルの設定などの機能を提供します。ユーザーのWebカメラからビデオストリームを収集して操作したり、3Dグラフィックスやオーディオサンプルを生成したりします。
  • 開発者がTwitterやFacebookなどの他のコンテンツプロバイダーのサイトに機能を組み込むことを可能にするサードパーティのAPI。
  • HTMLに適用して、サイトやアプリケーションの構築作業を加速できるサードパーティのフレームワークとライブラリ。

コアJavaScript言語が上記のツールとどのように異なるかについての詳細を示すことは、JavaScriptの簡単な紹介として、この記事の範囲外です。詳細については、MDNのJavaScript学習領域、およびMDNの他の部分を参照してください。

以下のセクションでは、コア言語のいくつかの側面を紹介し、いくつかのブラウザーAPI機能を試す機会も提供します。楽しんで！

言語の基本クラッシュコース

JavaScriptがどのように機能するかをよりよく理解するために、言語のコア機能のいくつかを説明しましょう。これらの機能はすべてのプログラミング言語に共通していることに注意してください。これらの基礎をマスターすれば、他の言語でのコーディングも有利に開始できます。

重要：この記事では、JavaScriptコンソールにサンプルコード行を入力して、何が起こるかを確認してください。 JavaScriptコンソールの詳細については、「ブラウザー開発ツールの検出」を参照してください。

変数

{{Glossary("Variable", "Variables")}}値を格納するコンテナです。まず、 var （あまり推奨されませんが、説明について詳しく説明します）またはletキーワードを使用して変数を宣言し、その後に変数に付けた名前を付けます。 

let myVariable;

注：行末のセミコロンは、ステートメントの終わりを示します。 1行でステートメントを区切る必要がある場合にのみ必要です。ただし、各ステートメントの最後にセミコロンを付けることをお勧めする人もいます。セミコロンを使用する必要がある場合と使用しない場合については、他にも規則があります。詳細については、 JavaScriptのセミコロンガイドを参照してください。

注：変数にはほぼすべての名前を付けることができますが、いくつかの制限があります。 （命名規則については、このセクションを参照してください。）不明な場合は、変数名をチェックして、それが有効かどうかを確認できます。

注：JavaScriptでは大文字と小文字が区別されます。これは、 myVariable同じではありませんmyvariable 。コードに問題がある場合は、ケースを確認してください。

注： varとlet違いの詳細については、 varとletの違いを参照してください。

変数を宣言した後、それに値を与えることができます： 

myVariable = 'Bob';

また、これらの両方の操作を同じ行で実行できます。 

let myVariable = 'Bob';

変数名を呼び出すことにより、値を取得します。 

myVariable;

変数に値を割り当てた後、コードの後半で値を変更できます。 

let myVariable = 'Bob';
myVariable = 'Steve';

変数は、異なるデータ型を持つ値を保持する場合があることに注意してください。

{{Glossary("Array")}}これは、1つの参照に複数の値を格納できる構造です。 let myVariable = [1,'Bob','Steve',10];
次のように配列の各メンバーを参照してください。
myVariable[0] 、 myVariable[1]など。

上記のすべての例も。

コメント

コメントは、コードとともに追加できるテキストのスニペットです。ブラウザは、コメントとしてマークされたテキストを無視します。 CSSと同じように、JavaScriptでコメントを書くことができます。 

/*
Everything in between is a comment.
*/

コメントに改行が含まれていない場合は、次のように2つのスラッシュの後ろにコメントを配置するオプションがあります。 

// This is a comment

条件付き

条件文は、式がtrueを返すかどうかをテストするために使用されるコード構造です。条件文の非常に一般的な形式は、 if ... elseステートメントです。例えば：

 let iceCream = 'chocolate';
if(iceCream === 'chocolate') {
  alert('Yay, I love chocolate ice cream!');
} else {
  alert('Awwww, but chocolate is my favorite...');
}

if( ... )内の式がテストです。これは、（上記のように）恒等演算子を使用して、変数iceCreamを文字列chocolateと比較し、2つが等しいかどうかを確認します。この比較がtrue返すtrue 、コードの最初のブロックが実行されます。比較が真でない場合は、 elseステートメントの後の2番目のコードブロックが代わりに実行されます。

関数

{{Glossary("Function", "Functions")}}再利用したい機能をパッケージ化する方法です。コード内で関数名を呼び出すと実行される関数として、コード本体を定義することができます。これは、同じコードを繰り返し記述する代わりに適しています。あなたはすでに以前に関数のいくつかの使用法を見てきました。例えば：

  1.  let myVariable = document.querySelector('h1');
  2. alert('hello!');

これらの関数document.querySelectorとalertは、ブラウザーに組み込まれています。

変数名のように見えるものがあり、その後に括弧が続く場合— () —それはおそらく関数です。関数はしばしばかかります{{Glossary("Argument", "arguments")}}：仕事をするために必要なデータのビット。複数の引数がある場合、引数は括弧内にコンマで区切ります。

たとえば、 alert()関数を使用すると、ブラウザウィンドウ内にポップアップボックスが表示されますが、表示するメッセージを関数に指示する引数として文字列を指定する必要があります。

独自の関数を定義することもできます。次の例では、2つの数値を引数として取り、それらを乗算する単純な関数を作成します。 

function multiply(num1,num2) {
  let result = num1 * num2;
  return result;
}

これをコンソールで実行してみてください。次に、いくつかの引数でテストします。例えば： 

multiply(4, 7);
multiply(20, 20);
multiply(0.5, 3);

注： returnステートメントは、ブラウザーに関数からresult変数を返して、使用できるようにするように指示します。関数内で定義された変数はそれらの関数内でのみ使用可能であるため、これが必要です。これは変数と呼ばれます{{Glossary("Scope", "scoping")}}。 （変数スコープの詳細をお読みください。）

イベント

Webサイトでの実際の対話には、イベントハンドラーが必要です。これらは、ブラウザでのアクティビティをリッスンし、それに応じてコードを実行するコード構造です。最も明白な例は、クリックイベントの処理です。これは、マウスで何かをクリックしたときにブラウザによって発生します。これを示すには、コンソールに次のように入力し、現在のWebページをクリックします。 

document.querySelector('html').onclick = function() {
    alert('Ouch! Stop poking me!');
}

イベントハンドラーを要素にアタッチする方法はたくさんあります。ここでは、{{htmlelement("html")}}要素onclickハンドラープロパティを匿名（つまり名前のない）関数と同じに設定します。この関数には、クリックイベントを実行するコードが含まれています。

ご了承ください

document.querySelector('html').onclick = function() {};

と同等です

let myHTML = document.querySelector('html');
myHTML.onclick = function() {};

ただ短いです。

イメージチェンジャーの追加

このセクションでは、JavaScriptおよびDOM API機能を使用して、2つの画像のいずれかの表示を交互に行う方法を学習します。この変更は、ユーザーが表示された画像をクリックすると発生します。

  1. サンプルサイトで紹介する画像を選択します。理想的には、画像は前に追加した画像と同じサイズ、またはできるだけ近いサイズになります。
  2. この画像をimagesフォルダに保存します。
  3. 画像の名前をfirefox2.pngに変更します。
  4. 以下のJavaScriptをmain.jsファイルに追加します。 （また、前の演習からHello world！ JavaScriptを削除します。）
     let myImage = document.querySelector('img');

myImage.onclick = function() {
    let mySrc = myImage.getAttribute('src');
    if(mySrc === 'images/firefox-icon.png') {
      myImage.setAttribute('src','images/firefox2.png');
    } else {
      myImage.setAttribute('src','images/firefox-icon.png');
    }
}
  5. すべてのファイルを保存し、ブラウザにindex.htmlをロードします。画像をクリックすると、他の画像に変わります。

これが起こったことです。あなたはあなたへの参照を保存しました{{htmlelement("img")}}myImage変数の要素。次に、この変数のonclickイベントハンドラープロパティを名前のない関数（「匿名」関数）と等しくしました。したがって、この要素がクリックされるたびに：

  1. このコードは、画像のsrc属性の値を取得します。
  2. コードは条件を使用して、 src値が元の画像のパスと等しいかどうかを確認します。
    1. そうである場合、コードはsrc値を2番目の画像のパスに変更し、他の画像を強制的に内部にロードします。{{htmlelement("img")}}素子。
    2. そうでない場合（つまり、すでに変更されている必要があります）、 src値は元のイメージパスにスワップバックし、元の状態に戻ります。

次に、ユーザーが最初にサイトにアクセスしたときに、ページタイトルをパーソナライズされたウェルカムメッセージに変更しましょう。このウェルカムメッセージは持続します。ユーザーがサイトを離れて後で戻った場合は、 Web StorageAPIを使用してメッセージを保存します。また、ユーザーを変更するオプション、つまりウェルカムメッセージも含まれます。

  1. index.html 、直前に次の行を追加します{{htmlelement("script")}}素子： 
    <button>Change user</button>
  2. main.jsで、次のコードをファイルの最後に、記述されているとおりに配置します。これは、新しいボタンと見出しへの参照を取り、それぞれの内部変数を格納します。
     let myButton = document.querySelector('button');

  3. 以下の機能を追加して、パーソナライズされた挨拶を設定します。これはまだ何もしませんが、これはすぐに変更されます。

  let myName = prompt('Please enter your name.');
  localStorage.setItem('name', myName);
  myHeading.textContent = 'Mozilla is cool, ' + myName;

    setUserName()関数には、 alert()と同様のダイアログボックスを表示するprompt() setUserName()関数が含まれていprompt() 。このprompt()関数は、 alert()以上のことを行い、ユーザーにデータの入力を求め、ユーザーが[ OK]をクリックした後にデータを変数に格納します。この場合、ユーザーに名前の入力を求めています。次に、コードはAPI localStorage呼び出しlocalStorage 。これにより、データをブラウザーに保存し、後で取得できます。 localStorageのsetItem()関数を使用して、 'name'というデータアイテムを作成および保存し、その値を、名前のユーザーエントリを含むmyName変数に設定します。最後に、見出しのtextContentを文字列に加えて、ユーザーの新しく保存された名前に設定します。

    if(!localStorage.getItem('name')) {
  setUserName();

nullのユーザー名？

例を実行して、ユーザー名の入力を求めるダイアログボックスが表示されたら、[キャンセル]ボタンを押してみてください。あなたはMozillaがかっこいい、nullと読むタイトルで終わるはずです。これは、プロンプトをキャンセルすると、値がnull設定されるために発生しnull 。 Nullは、値がないことを示すJavaScriptの特別な値です。

また、名前を入力せずに[ OK ]をクリックしてみてください。かなり明白な理由で、Mozillaがかっこいいというタイトルになってしまうはずです。

これらの問題を回避するために、ユーザーが空白の名前を入力していないことを確認できます。 setUserName()関数を次のように更新します。 

function setUserName() {
  let myName = prompt('Please enter your name.');
  if(!myName) {
    setUserName();
  } else {
    localStorage.setItem('name', myName);
    myHeading.innerHTML = 'Mozilla is cool, ' + myName;
  }
}

人間の言語では、これは次のことを意味しますmyNameに値がない場合は、最初からsetUserName()再度実行します。値がある場合（上記のステートメントが当てはまらない場合）、その値をlocalStorage保存し、見出しのテキストとして設定します。
