background-image CSSプロパティは、要素に1つ以上の背景画像を設定します。

{{EmbedInteractiveExample("pages/css/background-image.html")}}
このインタラクティブな例のソースは、GitHubリポジトリに保存されています。インタラクティブなサンプルプロジェクトに貢献したい場合は、 https：//github.com/mdn/interactive-examplesのクローンを作成して、プルリクエストを送信してください。

背景画像は、コンテキストレイヤーを積み重ねて描画されます。指定された最初のレイヤーは、ユーザーに最も近いかのように描画されます。

次に、要素の境界線がそれらの上に描画され、{{cssxref("background-color")}}それらの下に描かれています。ボックスとその境界線に対して画像がどのように描画されるかは、{{cssxref("background-clip")}}そして{{cssxref("background-origin")}}CSSプロパティ。

指定された画像を描画できない場合（たとえば、指定されたURIで示されるファイルをロードできない場合）、ブラウザーはそれをnone値と同じように処理します。

注：画像が不透明で、通常の状況では色が表示されない場合でも、Web開発者は常に{{cssxref("background-color")}}。ネットワークがダウンしている場合など、画像を読み込めない場合は、背景色がフォールバックとして使用されます。

構文

各背景画像は、キーワードnoneまたはとして指定されます。{{cssxref("<image>")}}値。

複数の背景画像を指定するには、カンマで区切って複数の値を指定します。 

background-image:
  linear-gradient(to bottom, rgba(255,255,0,0.5), rgba(0,0,255,0.5)),
  url('https://mdn.mozillademos.org/files/7693/catfront.png');

値

none
画像がないことを示すキーワードです。
<image>
です{{cssxref("<image>")}}表示する画像を示します。複数の背景がサポートされているため、コンマで区切って複数存在することができます。

アクセシビリティの懸念

ブラウザは、支援技術に背景画像に関する特別な情報を提供しません。スクリーンリーダーはその存在をアナウンスせず、したがってユーザーに何も伝えないため、これは主にスクリーンリーダーにとって重要です。画像にページの全体的な目的を理解するために重要な情報が含まれている場合は、ドキュメントで意味的に説明することをお勧めします。

例

背景画像を重ねる

星の画像は部分的に透明で、猫の画像の上に重ねられていることに注意してください。

HTML 

<div>
  <p class="catsandstars">
    This paragraph is full of cats<br />and stars.
  </p>
  <p>This paragraph is not.</p>
  <p class="catsandstars">
    Here are more cats for you.<br />Look at them!
  </p>
  <p>And no more.</p>
</div>

CSS 

p {
  font-size: 1.5em;
  color: #FE7F88;
  background-image: none;
  background-color: transparent;
}

div {
  background-image:
      url("https://mdn.mozillademos.org/files/6457/mdn_logo_only_color.png");
}

.catsandstars {
  background-image:
      url("https://mdn.mozillademos.org/files/11991/startransparent.gif"),
      url("https://mdn.mozillademos.org/files/7693/catfront.png");
  background-color: transparent;
}
